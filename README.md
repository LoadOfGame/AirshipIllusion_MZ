## 日本語

RPGツクールMZ用の飛空艇システムプラグインです。

### 概要

Airship Illusionは、RPGツクールMZでワールドマップ画像を使用した飛空艇システムを実装するプラグインです。

#### 主な機能

- ワールドマップ画像上での飛空艇の操作
- 離着陸システム
- 高度制限の設定
- カスタマイズ可能なパラメータ

### インストール

1. [Releases](../../releases)から最新版をダウンロード
2. `js/plugins`フォルダにプラグインファイルを配置
3. RPGツクールMZのプラグインマネージャーで有効化

### マニュアル

詳細な使い方は同梱の**AirshipIllusionManual.exe**を起動してご確認ください。
日本語・英語のマニュアルをブラウザで閲覧できます。

### 動作環境

## English

An airship system plugin for RPG Maker MZ.

### Overview

Airship Illusion implements an airship system using world map images in RPG Maker MZ.

#### Features

- Airship control on world map image
- Takeoff and landing system
- Altitude limit settings
- Customizable parameters

### Installation

1. Download the latest version from [Releases](../../releases)
2. Place the plugin files in the `js/plugins` folder
3. Enable in RPG Maker MZ Plugin Manager

### Manual

For detailed instructions, launch **AirshipIllusionManual.exe** included in the package.
You can view the manual in Japanese or English in your browser.
